CDN (unpkg.com)

unpkg.com
npmjs.com の全パッケージ(全バージョン？)が利用可能なCDN サービス
オリジナルホストはFly.io
CDN はCloudflare

URL ルール

unpkg.com/:package@:version/:file

@version を省略もしくは@latest をつけると最新バージョン
Vue3 https://unpkg.com/browse/vue@next/
@next をつけないとバージョン２
例
- https://unpkg.com/browse/vue@3.2.19/dist/vue.global.js
- https://unpkg.com/browse/vue@3.2.19/dist/vue.global.prod.js
react https://unpkg.com/browse/react/
例
- https://unpkg.com/react@16.7.0/umd/react.production.min.js
- https://unpkg.com/react-dom@16.7.0/umd/react-dom.production.min.js
jQuery https://unpkg.com/browse/jquery/
例
- https://unpkg.com/jquery@3.6.0/dist/jquery.min.js
- https://unpkg.com/jquery@3.6.0/dist/jquery.slim.min.js
