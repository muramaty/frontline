HTMLテンプレート

<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="utf-8">
  <title>タイトル</title>
  <meta name="description" content="ディスクリプション">
  <meta name="keywords" content="キーワード" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <!-- script src="app.js"></script -->
</head>
<body>
  <div id="header">
    サイト名
    <nav>内部リンク</nav>
  </div>
  <main>
    <h1>見出し１</h1>
    <time datetime=”2021-09-03” itemprop=”modified”>2021年9月3日</time>
    <p>ページの説明</p>
    <section>
      <h2>見出し２</h2>
      <p>見出し２の説明</p>
      <div>
        見出し２の内容
      </div>
    </section>
  </main>
  <div id="footer">
    フッター
  </div>
</body>
</html>

タグの構成、アウトライン

コンテンツの内容、CSS、広告表示、レスポンシブ対応などによりますが、
HTMLの構成を決めるときに以下の点を考慮しています。
  • header、footerタグは文章構成での意味を持つタグなので使用しない(使いどころがわからない)。レイアウトの上部、下部に位置するヘッダー部分、フッター部分はdiv を利用
  • メインコンテンツ部分はmainタグに記述
  • 1つの内容をsectionタグ、h2タグに対応させ、必要に応じて複数記述
  • h1タグの近くにtime タグをつけて更新日を記述
