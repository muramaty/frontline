yarnコマンド

yarn init新規パッケージ、プロジェクトを作成
yarn install依存するパッケージをインストール
yarn add パッケージ(@version) (--dev)パッケージをインストール(@バージョン指定)
yarn up パッケージ(@version)パッケージをアップデート(@バージョン指定)
yarn removeパッケージを削除
yarn set version version|latestyarn自体をアップデート

リンク: yarn cli

gitリポジトリから直接add する
yarn add <package name>@git+ssh://<git remote URL>
